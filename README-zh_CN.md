## 软件的设计原则：

1. 数据完全本地存储

不联网，无需担心数据的安全性和隐私问题。提供基于手机原始GPS、网络基站定位的空间轨迹记录功能。

2. 最小依赖原则

专注于提高定位服务稳定性，轨迹记录，本地统计分析和数据呈现。

- 不集成地图；
- 不联网；
- 无广告；
- 无推荐性引导；
